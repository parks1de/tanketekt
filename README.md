# TankeTekt v.demo-002
Norsk nettlesar-basert planteikning. Opne index.html i nettlesar.
Ingen installasjon nødvendig.
Funksjonar: teikn rom og veggar, snapping, zoom/pan, dører/vindauge, romtypar, TEK17-åtvaringar, angre/gjer om, lagre/last inn JSON, eksport PNG + vektor-PDF.
Bygd av 6 Claude API-agentar.
